# ❄️ Shirayuki

**The modern, beautiful anime streaming platform — built with React, Vite, and Tailwind CSS.**

![Shirayuki Banner](public/logo/shirayuki1.png)

---

## ✨ Features

- 🚀 Blazing-fast UI with Vite + React 19
- 🎨 Gorgeous, responsive design (Tailwind CSS)
- 🔎 Advanced anime search, genres, categories, and filters
- 📅 Seasonal schedules, A-Z lists, and studio/producer pages
- 📝 User profiles, favorites, and watch progress
- 📺 High-quality streaming with multiple servers
- 🔐 Secure authentication and profile management
- 🌙 Dark mode, smooth animations, and more!

---

## 🛠️ Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/your-username/shirayuki.git
   cd shirayuki
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env and set your API base URL
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open in your browser**
   ```
   http://localhost:5173
   ```
